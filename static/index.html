Syntax rules

Common syntax

|- or ⊢thesis sign
A, B, ..., Zformulas
,to separate formulas
( and )around a sub-formula

Example: A |- B,C

Linear logic syntax

^negation (e.g. A^)
* or ⊗tensor (multiplicative conjunction)
|par (multiplicative disjunction)
&with (additive conjunction)
+ or ⊕plus (additive disjunction)
1one (multiplicative conjunction unit)
bot or ⊥bottom (multiplicative disjunction unit)
top or ⊤top (additive conjunction unit)
0zero (additive disjunction unit)
-o or ⊸lollipop (linear implication)
!"of course"
?"why not"

Example: |- A^,A|B,!C

Classical logic syntax

You can also use classical logic, but you can't mix it with linear logic syntax.

~ or ¬negation
/\conjunction
\/disjunction
truetrue
falsefalse
-> or => or →implication

Example: A,~B,(A\/C)/\(A\/B) |-
